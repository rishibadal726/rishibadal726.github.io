Section 1

1.1: Compare this behavior to that of the while loop. What are the similarities? What are the differences?

Similarities: Both of them tests a condition, they both have Jump instructions, and they have an alternative when the condition is false. Differences: The repeat-while loop starts with a statement and executes it first then it checks to see if the condition is true or false, while the while loop starts with the condition first and check to see if it's true or false to see if the statements in the body of the loop will be executed.

1.2: What are the minimum number of times that this loop will execute?

The minimum numbers of times that this loop will execute it 1 time.

1.3: What is the purpose of the Jump instruction after the consequent?

The purpose of the Jump instruction after the consequent is to make a jump back to the statements at the beginning of the Repeat-While Loop.

Section 2

2.1: Are you able to implement the same functionality found in a while loop using a repeat-while loop, without changing the condition or adding a conditional? Explain and provide an example.

You are able to implement the same functionality found in a while loop using a repeat-while loop, without changing the condition or adding a conditional as long as the first intial value of the variable is true in the conditions that follow, otherwise you can't implement the same functionality found in a while loop using a repeat-while loop. For example: when x = 8 then the while loop and repeat-while loop will have the same output due to the condition being true, but when x = 3 then the while loop and repeat-while loop will not have the same output due to the condition being false.
var x = 8
repeat {
print("consequent")
x -= 1
} while (x > 4)
print("alternative")
